 Django_Simple_CRUD-Operations
 
 Languages Handled :
 
 Python
 
 Framework- Django

 - > Able to Add Movie Title (Id, Title, Year)
 
 - > Able to See Entire movie list in landing page
 
 - > Able to Delete Movie with their specific Id
